Lists

Unordered Lists:

In unordered lists, the order doesn't matter. (ex. a shopping list)

  • Element 1
  • Element 2
  • Element 3
  • Element 4

Ordered Lists:

In ordered lists, the order matters, and it will be numbered. (ex. a set of instructions)

  1. Element 1
  2. Element 2
  3. Element 3
  4. Element 4
